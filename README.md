# GrappleManX



* Throwaway project from a random concept I had in middleschool that I never knew how to do
* Decided to just see how far I could get solo developing every aspect to showcase my abilities as a gameplay programmer
* Tried to cover every single role in development, dipped my toes into everything a little.
* Focused on a high movement character controller, just a conglomerate of various movement mechanics from a bunch of games.
* Definitely not polished at all, just a proof to myself that if a project required it I could support any role that needed me.





Features:

\-High speed character controller with

&#x09;-Sliding (With slide jumping)

&#x09;-Wall Running

&#x09;-Grapple Hook mechanics (pull, rope wrapping)

&#x09;-Dynamic mantling system on ledges

&#x09;

\-3D Rigged and animated models of some bullshit characters I came up with in like middleschool

&#x09;-I wanted them to be interactable with the environment because that physics jazz is really annoying.



\-2 Rough cut levels, with a chase sequence at the end of the second one



\-1 Really shitty tutorial



\-A dialogue system with a really cringe narrative written by a 14 year old (I did not want to do any writing this semester so its all from back then)



\-A menu system



\-Music system with (1) self-composed song
